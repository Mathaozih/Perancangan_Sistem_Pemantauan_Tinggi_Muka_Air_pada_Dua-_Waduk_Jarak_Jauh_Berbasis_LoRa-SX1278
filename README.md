# Perancangan_Sistem_Pemantauan_Tinggi_Muka_Air_pada_Dua-_Waduk_Jarak_Jauh_Berbasis_LoRa-SX1278

Project ini merancang sistem pemantauan tinggi muka air pada dua waduk yang berada di lokasi terpisah menggunakan LoRa SX1278. Sistem ini akan mengirimkan data ketinggian muka air masing-masing ke stasiun penerima dan menampilkan data pada lCD dan memberikan notifikasi setiap keadaan ke telegram bot.
